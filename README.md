# vp-labtask

A collection of Visual Programming lab tasks built with **HTML**, **CSS**, and **Bootstrap 5**. This repository contains five mini front-end projects demonstrating various UI layouts and components.

---

## 📁 Project Structure

```
vp-labtask/
├── calculator/
│   └── index.html        # iOS-style calculator UI
├── cv/
│   └── index.html        # Personal CV / Resume page
├── timetable/
│   └── index.html        # Weekly class timetable
├── fb-home/
│   └── index.html        # Facebook homepage clone
└── fb-auth/
    ├── login.html         # Facebook login page clone
    └── signup.html        # Facebook signup page clone
```

---

## 🗂️ Projects Overview

### 🧮 Calculator
An iOS-inspired calculator UI featuring:
- Dark theme with orange operator buttons
- Circle-shaped buttons with proper grid layout
- Special wide zero button styled to match Apple's calculator

### 📄 CV / Resume
A two-column personal resume page for **Abdul Hadi** featuring:
- Profile picture, contact info, skills, languages, and hobbies in a dark sidebar
- Profile summary, projects, and education in the main content area

### 📅 Timetable
A weekly class schedule for **BSCS-IV-C (Shift-I)** featuring:
- Full 5-day timetable (Mon–Fri) across 9 time slots
- Color-coded cells for lectures and lab sessions (spanning multiple columns)
- Subjects: Software Engineering, Computer Networks, Visual Programming, Computer Organization & Assembly Language, Multivariable Calculus, and more

### 🏠 Facebook Home Page
A pixel-faithful Facebook newsfeed clone featuring:
- Fixed top navbar with search, nav icons, and action buttons
- Left sidebar with navigation links and shortcuts
- Stories, post composer, and sample news feed posts
- Right sidebar with sponsored ads and online contacts

### 🔐 Facebook Login & Signup Pages
Two-page Facebook authentication UI clone featuring:
- **Login page** — email/phone + password form with "Forgotten password?" and "Create new account" links
- **Signup page** — full registration form with name, contact, password, date of birth, and gender fields
- Pages are linked to each other for navigation flow

## 🚀 How to Run

No build tools or dependencies required. Simply open any `index.html` or `login.html` / `signup.html` file directly in your browser:

```bash
# Clone the repo
git clone https://github.com/your-username/vp-labtask.git

# Navigate to any project folder and open in browser
cd vp-labtask/calculator
open index.html
```

Or use the **Live Server** extension in VS Code for a better development experience.
